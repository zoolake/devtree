## ****프로젝트 소개****


멘토링을 통해 효과적인 개발자 스터디 & 프로젝트를 도와주는 서비스

![image](/uploads/8bee1ad0020a3d43187dae32e511400a/image.png)

### DEVTREE의 뜻?

- 멘토를 양분삼아 자라나는 주니어 개발자들
- 트리 구조처럼 연결된 개발자 커넥션
- 나뭇가지처럼 뻗어나가는 지식들

### 환경 버전

1. **이슈관리**: Jira
2. **형상관리** : Gitlab
3. **커뮤니케이션** : Mattermost, Discord, Notion
4. **개발 환경**
    - **OS**
        - Windows 10
    - **IDE**
        - Intellij IDEA community Edition 2021.3.1
        - Visual Studio Code 1.63.2
        - Figma, illustrator
    - **Database**
        - MySQL Workbench
    - **Server**
        - AWS EC2 - Ubuntu 20.04 LTS (GNU/Linux 5.4.0-1018-aws x86_64)
    - **Browser**
        - Chrome 97.0.4692.71 64bit
        - Microsoft Edge 97.0.1072.69 64bit
5. **상세내용**
    - **Backend**
        - Java (Open-JDK Zulu 8.33.0.1 / 11.0.X)
    - **Frontend**
        - HTML5, CSS3, JavaScript(ES6)
        - React 17.0.2, Reux
        - Node.js 16.13.0
- node: `16.13.0`
- java: `1.8.xx` ***(java 8 이상 : 스트림, 람다)***
- mysql: `8.0이상` / `5.7이상`
- react : 최신버전(`17.0.2`)
- spring-boot
- gradle
- WebRTC관련 서버
- browser: Chrome
- OS: Windows 10
